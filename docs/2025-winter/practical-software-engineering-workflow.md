## Git vs Git providers

### Git

- Version control system
- Acts on code repositories

Basic features of git:

- Repositories
- Commits
- Branches
- Merging

### Github

- Hosts code repositories
- Supporting functionality to assist with development

Basic features of Github:

- Pull requests
- Code reviews
- Comments
- Github Actions
- Permissions
